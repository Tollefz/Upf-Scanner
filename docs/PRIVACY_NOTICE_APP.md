# Privacy Notice - In-App Tekst

## Versjon 1 (Kort, for første gang)

```
Når du rapporterer et ukjent produkt, sender vi:
• Produktkode (GTIN)
• Produktbilde (hvis du tar bilde)
• Tekstnotat (valgfritt)

Vi bruker en anonym teknisk identifikator for å beskytte mot misbruk. Vi samler ikke navn, e-post eller lokasjon.

Dataene brukes kun til å forbedre produktdatabasen.
```

## Versjon 2 (Litt mer detaljert)

```
Hva sendes når du rapporterer?

• Produktkode (GTIN) – for å identifisere produktet
• Produktbilde (hvis du tar bilde) – kun for produktidentifisering
• Tekstnotat (valgfritt) – for å beskrive produktet

Vi samler ikke:
• Navn, e-post eller telefonnummer
• GPS-lokasjon
• Bruker-ID

Vi bruker en anonym teknisk identifikator for å beskytte mot misbruk.

Dataene brukes kun til å forbedre produktdatabasen. Les mer i vår privacy policy.
```

## Versjon 3 (Ekstra kort, for knapp)

```
Vi sender kun produktkode og bilde (hvis valgt). Ingen persondata samles. Se privacy policy for mer info.
```

## Anbefalt Implementasjon

**Vis privacy notice:**
- Første gang bruker sender rapport (one-time)
- Eller ved behov i UnknownProductScreen

**Plassering:**
- Over "Rapporter" knappen
- Eller som info-ikon med modal/drawer
