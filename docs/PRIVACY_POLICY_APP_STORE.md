# Privacy Policy - App Store Connect

```

• Produktkode (GTIN)
• Produktbilde (hvis du velger å ta bilde)
• Valgfritt tekstnotat om produktet

• Navn, e-postadresse eller telefonnummer

• Vi bruker anonyme tekniske identifikatorer for misbrukskontroll
• Data deles ikke med tredjeparter

Du har rett til å be om innsyn, retting eller sletting av dine data. Kontakt oss på [e-post].

Kontakt

```

## Engelsk Versjon (for App Store)

```
Privacy Policy for [App Name]

What data do we collect?
When you report unknown products, we collect:
• Product code (GTIN)
• Product image (if you choose to take a photo)
• Optional text note about the product

We do NOT collect:
• Name, email address, or phone number
• GPS location or other location information
• User ID or account information

Why do we collect data?
Data is used solely to improve the product database so more users can find information about food products.

How do we protect your data?
• Images are stored in encrypted storage
• We use anonymous technical identifiers for abuse prevention
• Data is not shared with third parties
```

## Utvidet Versjon (Hvis nødvendig)

```
Personvernpolicy for [App-navn]

1. Innledning
[App-navn] er en matvare-skanner-app som hjelper brukere med å finne informasjon om produkter. Denne personvernpolicyen forklarer hvilke data vi samler og hvordan vi bruker dem.

2. Hvilke data samler vi?
Når du rapporterer et ukjent produkt via appen, samler vi følgende:

Påkrevd data:
• Produktkode (GTIN/EAN) – nødvendig for å identifisere produktet

Valgfritt data (du velger selv):
• Produktbilde – hvis du tar et bilde av produktet
• Tekstnotat – hvis du ønsker å beskrive produktet

Teknisk data (automatisk):
• Tidsstempel for når rapporten ble sendt
• App-versjon og plattform (iOS/Android) – for feilsøking
• Språkinnstilling (f.eks. "da-DK") – for å forstå kontekst
• Anonym teknisk identifikator – for å beskytte mot misbruk (rate limiting)

Vi samler IKKE:
• Navn, e-postadresse eller telefonnummer
• GPS-lokasjon eller annen posisjonsinformasjon
• Bruker-ID, kontoinformasjon eller innlogging
• IP-adresse (bruker signed URLs som ikke logger IP)
• Annen persondata

3. Hvorfor samler vi data?
Dataene brukes utelukkende til følgende formål:
• Å forbedre produktdatabasen med manglende produkter
• Å gi bedre produktopplysninger til alle brukere
• Å beskytte mot misbruk (via anonym teknisk identifikator)

Dataene brukes IKKE til:
• Reklame eller markedsføring
• Profilering av brukere
• Deling med tredjeparter
• Annen kommersiell bruk

4. Hvordan lagrer vi data?
• Bilder lagres i kryptert, privat lagring (Supabase Storage)
• Databasen er beskyttet med Row Level Security (RLS)
• Teknisk identifikator er hashet og anonymisert
• Data lagres i EU (Supabase)

5. Hvor lenge beholder vi data?
• Rapporter beholdes inntil produktdata er beriket (typisk 1-2 år)
• Data kan anonymiseres eller slettes etter 2 år
• Du kan be om sletting når som helst (se punkt 7)

6. Deling med tredjeparter
Vi deler ikke data med tredjeparter. Produktdata kan deles via åpne API-er (f.eks. Open Food Facts) for å forbedre produktdatabasen globalt, men dette skjer uten persondata.

7. Dine rettigheter (GDPR)
Du har følgende rettigheter:
• Innsyn – be om å se hvilke data vi har om deg
• Retting – be om retting av feilaktige data
• Sletting – be om sletting av dine data
• Dataportabilitet – be om utlevering av dataene
• Klage – klage til Datatilsynet

For å utøve disse rettighetene, kontakt oss på [e-post].

8. Sikkerhet
Vi tar sikkerheten på alvor:
• All dataoverføring er kryptert (HTTPS)
• Bilder lagres i privat, kryptert lagring
• Vi bruker signed URLs for bilde-upload (kort levetid)
• Ingen direkte tilgang til lagring fra appen
• Rate limiting for å beskytte mot misbruk

9. Endringer i personvernpolicy
Vi kan oppdatere denne policyen. Vesentlige endringer vil bli kommunisert via appen.

10. Kontakt
For spørsmål om personvern eller for å utøve dine rettigheter:
E-post: [e-post]
Postadresse: [adresse]

Sist oppdatert: [dato]
```
